    项目经历

    1. 全员纷享（客户圈）APP

      混合应用，使用 MSUI 实现 UI 层，使用 artTemplate 渲染页面，使用 Ponto 与 桥接 Native，提交 70% 的业务代码。

      贡献：2017 年初提出使用 Vue 替代 artTemplate 实现页面渲染和交互，使用 mint-ui 替代 MSUI 实现 UI 层，并提供示例页面。

    2. 70度微商城卖家平台（移动端）

      使用 MSUI (阿里巴巴国际UED前端团队出品的移动端 UI库) 实现的单页应用，提交 80% 的业务代码。

    3. 70度微商城 - 商品购买

      以 OOP 方式动态实现商品展示、购买、订单和查询物流页面，使用 laytpl 渲染页面，独立完成。

      贡献：解决一个极其麻烦的跨域问题，原本这个项目是准备做成嵌入H5场景预览 iframe 的商品购买页面，跟场景预览不在同一域名，这就导致在微信浏览器内打开H5场景购买商品需要把商品购买页面的微信配置信息发送给H5场景做微信授权等操作。解决方案是使用 HTML5 跨域通信 API - window.postMessage()

    4. 70度H5场景制作平台

      基于 AngularJS 1.x 的单页应用开发。

      贡献：发现很多用户打开H5场景是因为好奇或其它原因，手机打开场景后并没有详细查看所有页面内容，大量未被用户看到的图片资源被下载（浪费流量）并渲染到场景页面（初始化缓慢），因此提出场景图片资源懒加载和预加载，并提供基本方案和代码。方案实现之后，针对此类用户打开场景流量消耗减少 80%！并且页面初始化渲染速度得到小幅提升。
